Project Details

=> મેં બધા જ પ્રેક્ટિકલ્સ (1 to 21) ને એક Modular Menu-Driven System માં બનાવ્યા છે. 
   આ પ્રોજેક્ટમાં Logic, UI અને Data ને અલગ-અલગ ફાઈલોમાં વહેંચવામાં આવ્યા છે:

1> main.c       : પ્રોગ્રામ શરૂ કરવા માટે.
2> globals.h    : બધા Structures અને Unions માટે.
3> assets/      : આ ફોલ્ડરમાં પ્રેક્ટિકલનું Logic (modules.c) અને UI (ui.c) રાખેલું છે.

=>  પ્રોગ્રામ કેવી રીતે રન કરવો? (How to Run?)
    પ્રોજેક્ટ રન કરવા માટે ૨ રસ્તા છે:
        રીત ૧: ફોલ્ડરમાં આપેલી b.bat ફાઈલ રન કરવી. (Easy Way)
        રીત ૨: ટર્મિનલમાં આ કમાન્ડ ટાઈપ કરવો:
        gcc main.c assets/modules.c assets/ui/ui.c -o main && main.exe
